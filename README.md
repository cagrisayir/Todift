# Todift: To-do App 

This is a To-do app made with Swift, SwiftUI, and Firebase by cagrisayir. 

### Technologies and features that I used: 
* Firebase Auth
* Firestore
* Onboarding
* Splash Screen (Animated)
* Google Admob
* CocoaPods
